# Hand-detection-Project
This project demonstrate real-time hand detection and tracking using computer vision. 
The system detects an user's hand through a webcam and identifies hand landmarks (finger joints and palm point.

## Project Overview
The objective of this project is to 
- Detect hands from a live webcam feed
- identify finger and palm landmarks
- Track hand movement in real time

  This type of technology is used in
  - Gesture recorgnition
  - sign language interpretation
  - Virtual relity (VR)
  - Human computer vision
  - Gaming interfaces
 
    ## Technology used
    . Python
    . OpenCV - For video capture and image processing
    . MediaPipe: For hand langmark detection
